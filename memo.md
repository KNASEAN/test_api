# [git commit error](https://zenn.dev/ikkik/articles/475924c5c97718)
原因
git設定にuser.nameとuser.emailが存在しない。

```
ユーザ名の設定
git config user.name [ユーザ名]

 メールアドレスの設定
git config user.email [メールアドレス]
```
